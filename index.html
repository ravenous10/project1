SI. Tags Description Example
01 <html> </html> The Hypertext Transfer Protocol (HTTP) is an application protocol for distributed, collaborative, hypermedia information systems.
HTTP is the foundation of data communication for the World Wide Web.		 <!DOCTYPE html>
<html>
<head>
<title> </title>
</head>
<body>
</body>
</html>
02 <head></head> The <head> element is a container for all the head elements.
The <head> element can include a title for the document, scripts, styles, meta information, and more.		 <head>
<title> </title>
</head>
03 <title> </title> The <title> tag is required in all HTML documents and it defines the title of the document. <title>This is title </title>
04 <body> </body> The <body> tag defines the document's body.
The <body> element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.		 <body> This is Body </body>
05 <p> </p> The <p> tag defines a paragraph.
Browsers automatically add some space (margin) before and after each <p> element.
The margins can be modified with CSS (with the margin properties).		 <p>This is paragraph. </p>
06 <a> </a> HTML links are defined with the <a> tag <a href="http://www.google.com" > </a>
07 <img> HTML imagse are defined with the <img> tag.
The source file (src), alternative text (alt), width, and height are provided as attributes		 <img src="image source">
08 <span> </span> The <span> tag is used to group inline-elements in a document.
The <span> tag provides no visual change by itself.
The <span> tag provides a way to add a hook to a part of a text or a part of a document.		 <span style="color: green"> green
09 <link> The <link> tag defines a link between a document and an external resource.
The <link> tag is used to link to external style sheets.		 <link rel="stylesheet" type="text/css">
10 <br> The <br> tag inserts a single line break.
The <br> tag is an empty tag which means that it has no end tag.		 This text contains<br>a line break.
11 <h1> to <h6> The <h1> to <h6> tags are used to define HTML headings.
<h1> defines the most important heading.
<h6> defines the least important heading.		 <h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>
12 <strong> The <strong> tag is a phrase tag. It defines important text. <strong>Strong text</strong>
13 <th> The <th> tag defines a header cell in an HTML table. <table> <tr> <th>Month</th> <th>Savings</th> </tr></th>
14 <td> The <td> tag defines a standard cell in an HTML table. <table> <tr> <td>Cell A</td> <td>Cell B</td>
